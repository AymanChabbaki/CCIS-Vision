### State Management Strategy
- **Local State**: `useState` for component-specific state
- **Global State**: Zustand for auth, user preferences
- **Server State**: TanStack Query for API data with caching
- **Form State**: React Hook Form for complex forms

### Styling Strategy
- **Tailwind CSS** for 95% of styling
- **Inline styles** for dynamic values only
- **CSS modules** if component-specific styles needed
- **No styled-components or CSS-in-JS**
